## 第一步：找到你的登录信息

搬瓦工买了怎么用，先从拿到登录信息开始。

买完之后，搬瓦工会往你注册的邮箱发一封邮件，里面有三样东西：**IP 地址、SSH 端口、root 密码**。这三样是后续所有操作的钥匙，记好。

找不到邮件怎么办？登录搬瓦工账号后台，点 Services → My Services，找到你买的那台 VPS，点右边的 **KiwiVM Control Panel** 进控制面板。主界面第一屏就能看到 IP 和 SSH 端口。至于 root 密码——KiwiVM 里看不到，只有购买邮件里有。如果找不到，就在控制面板左侧找 **Root password modification**，先把 VPS 关机（点 stop），再点 **Generate and set new root password**，重新生成一个，然后记下来。

重置密码这一步有个坑：**一定先关机再重置**，不然会失败。生成的新密码只显示一次，务必抄下来或者截图。

## 第二步：用 SSH 连上服务器

有了 IP、端口、密码，就可以远程登录了。具体怎么操作，看你用哪个系统：

### Windows 用户

推荐 **Termius**，免费、界面清晰、跨平台都能用。下载装好之后，新建一个 Host，填入 IP 地址、端口（搬瓦工的 SSH 端口不是默认的 22，是随机生成的，要用你自己面板里看到的那个）、用户名填 root、密码填刚才拿到的 root 密码。点连接，看到命令行提示符出来就算成功了。

不想装软件的话，用 PowerShell 或者 Windows 自带的命令提示符也行，输命令：


ssh root@你的IP地址 -p 你的端口号


回车，再输密码。密码输的时候屏幕上不会显示任何字符，这是正常的，盲打完直接回车就行。

### Mac / Linux 用户

打开终端，直接输：

bash
ssh root@你的IP地址 -p 你的端口号


Mac 不需要装任何额外软件，系统自带的终端就够。

## 第三步：搞懂 KiwiVM 控制面板

搬瓦工买了怎么用，KiwiVM 是绕不开的东西。这是搬瓦工自研的控制面板，不用第三方面板，功能挺全的。

进入方式：搬瓦工账号后台 → Services → My Services → KiwiVM Control Panel。

主要功能盘点：

**Main Controls（主控区）**：可以看 VPS 当前状态、IP、端口、流量用量。四个操作按钮：start 开机、stop 关机、reset 重启、force stop 强制关机。正常操作用前三个，force stop 别轻易点，有数据丢失风险。

**Install new OS（重装系统）**：支持 CentOS、Debian、Ubuntu、Rocky Linux 等多个版本，还有带 BBR 加速的版本可以选。重装前必须先 stop 关机，不然会失败。重装后系统里所有数据清空，新密码会在重装完成页面显示一次，要记好。

**Migrate to another DC（迁移机房）**：这个是搬瓦工比较特别的功能。部分套餐支持在后台直接切换机房，比如你买的是 CN2 GIA-E 套餐，当前用的是洛杉矶 DC6，但感觉速度一般，可以切到日本软银 JPOS_1 或者荷兰 EUNL_9 试试。迁移会换 IP，数据也会迁移过去，一般需要 10-15 分钟，完成后邮件通知。

**Snapshots（快照）和 Backups（自动备份）**：这两个是免费的，用好了能救命。快照是手动创建当前系统的完整镜像；自动备份是搬瓦工定期帮你备份，出事了能回滚。部署完重要服务之后，养成打快照的习惯。

**Detailed Statistics（详细统计）**：查看 CPU 占用、流量使用、网络情况，对排查卡顿很有用。

## 新手常见问题

**Q：买完没收到邮件怎么办？**
先查垃圾邮件箱。还是没有的话，登录搬瓦工账号后台，点 Support → Email History，所有发过的邮件都能在里面找到。

**Q：连 SSH 总是超时连不上，是被封了吗？**
搬瓦工 IP 在国内有可能被封。判断方法：在 KiwiVM 面板里能正常看到 VPS 状态、VPS 还在运行，但就是 SSH 连不上，大概率是 IP 被封了。可以试试通过 KiwiVM 里的 **Root shell - basic** 在线终端看看服务器有没有问题，或者去 Migrate to another DC 换一个机房（会换 IP）。

**Q：流量快用完了怎么办？**
KiwiVM 面板 Main Controls 里能看到当前流量使用情况。搬瓦工流量是按月重置的，重置日期是你购买套餐的对应日期。如果月中用完了，可以在 KiwiVM 里花钱买额外流量，也可以等下个月重置。

**Q：搬瓦工支持退款吗？**
支持。官方政策是 30 天内可以申请退款，不满意退全额。走 Support 提工单申请。实际用下来退款流程挺顺畅的，不用担心跑路风险。

**Q：VPS 里的数据怎么备份？**
两种方式：KiwiVM 面板里的 Snapshots（快照）是手动备份，想备就备；Backups（自动备份）是搬瓦工定期自动做的，免费的，放心用。重要数据养成定期打快照的习惯。
